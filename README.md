**Веб войс-чат**

Простенький проект созданный на основе дискорд с помощью Peer JS.
Нечего больше сказать, просто заходите в комнату и разговариваете.
Конечно же могут возникать ошибки, проект не доведен до идеала. 
Проект показан показать возможности Peer JS и не более.


<hr>

**Не забудьте**
Заменить домены на ваш в
<ul>
    <li>env файлы</li>
    <li>peer js (server.js)</li>
    <li>На клиентской части в папке peer (index.ts)</li>
</ul>
