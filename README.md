**Веб войс-чат**

Веб-войс-чат — простенький проект, созданный на основе Discord с помощью PeerJS. Нечего больше сказать: просто заходите в комнату и разговариваете. Конечно же, могут возникать ошибки, проект не доведен до идеала. Проект показан, чтобы показать возможности PeerJS и не более.

<hr>

**Не забудьте**
Заменить домены на ваш в
<ul>
    <li>env файлы</li>
    <li>peer js (server.js)</li>
    <li>На клиентской части в папке peer (index.ts)</li>
</ul>
